    The purpose of this project was to analyze how weather changes as you get closer to the equater. We needed to execute API calls in order to get information on over more than 500 cities' weather characteristics.

    All graphics you will be observing were assembled using MatplotLib.

    The studies were executed on Temperature, Humidity, Cloudiness and WindSpeed.
